   - 4 gauges for: Convention Coverage, Type Safety, Test Coverage Proxy, Import Graph Health

     - `viewBox="0 0 140 90"`, width 140, height 90
     - Background arc: full semicircle in `#334155` (tertiary bg)
     - Fill arc: proportional to percentage, colored by status:
       - >=75: `#22C55E` (accent/green)
       - 50-74: `#F59E0B` (warning/yellow)
       - <50: `#EF4444` (danger/red)
     - Center text: percentage in Fira Code 28px semibold + letter grade below in 14px
     - Label below SVG: dimension name in Fira Sans 12px muted
   - SVG arc math:
     ```
     radius = 60, strokeWidth = 12, cx = 70, cy = 70
     startAngle = PI (180deg left)
     endAngle = PI - (percent/100 * PI) (sweeps right)
     bgPath: M (cx - radius) cy A radius radius 0 1 1 (cx + radius) cy
     fillEndX = cx + radius * cos(endAngle)
     fillEndY = cy - radius * sin(endAngle)
     largeArc = percent > 50 ? 1 : 0
     fillPath: M (cx - radius) cy A radius radius 0 largeArc 1 fillEndX fillEndY
     ```
   - Both arcs use `stroke-linecap="round"` for rounded ends

4. **Historical trend line chart (D-27, D-28):**

   - SVG viewBox scaled to fit container width and ~250px height

   - 4 line series (one per dimension): each a `<polyline>` with points mapped from history data
