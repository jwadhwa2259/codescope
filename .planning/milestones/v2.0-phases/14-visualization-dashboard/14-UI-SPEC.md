## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, icon-to-label spacing |
| sm | 8px | Compact element spacing, tooltip padding, status bar padding |
| md | 16px | Default element spacing, card padding, panel content margin |
| lg | 24px | Section padding, drawer section gaps, gauge chart margins |
| xl | 32px | Layout gaps, panel top/bottom padding |
| 2xl | 48px | Major section breaks, command center card grid gap |
| 3xl | 64px | Page-level spacing (not used -- panels fill viewport) |

Exceptions:
- Sidebar width: 56px (D-13) -- not on the 8-point grid, acceptable for fixed chrome
- Drawer width: 480px (D-37) -- fixed panel overlay
- Touch targets: minimum 32px for all clickable elements (developer tool, not mobile)
- Node sizes: 4px to 24px range for sigma.js graph nodes (D-19) -- driven by centrality data, not spacing tokens

**Source:** Default 8-point scale + CONTEXT.md D-13, D-19, D-37

## Typography

| Role | Size | Weight | Line Height | Font Family |
|------|------|--------|-------------|-------------|
| Body | 14px | 400 (regular) | 1.5 | Fira Sans |
| Label / Caption | 12px | 400 (regular) | 1.4 | Fira Sans |
| Heading | 18px | 600 (semibold) | 1.2 | Fira Sans |
| Data Value | 14px | 400 (regular) | 1.0 | Fira Code |
| Data Value Large | 28px | 600 (semibold) | 1.0 | Fira Code |

Font sizes declared: 12, 14, 18, 28 (4 sizes).
Font weights declared: 400 (regular) + 600 (semibold).

**Rationale:**
- 14px body: dense data dashboard with many values on screen simultaneously
- 12px label: secondary text (tooltip content, status bar items, axis labels)
- 18px heading: panel titles, drawer section headers
- 28px data value large: gauge center percentages, hero metric numbers
- Fira Code for all numeric/path data to ensure alignment and readability
- Fira Sans for all UI chrome, labels, and headings

**Source:** CONTEXT.md D-02, defaults for data-dense dashboard

## Interaction Contract

### Global Interactions

| Interaction | Behavior |
|-------------|----------|
| Panel switching (sidebar click) | Instant swap -- no transition animation between panels. Active indicator slides to new position (200ms ease-out). |
| Panel switching (keyboard 1-5) | Same as sidebar click. Keys `1` through `5` map to Graph, Heatmap, Trends, Blast, Command respectively. |
| Hover on sidebar icon | Show tooltip with panel name (right of icon, 200ms delay) at all viewport widths, including collapsed sidebar below 1200px. Each sidebar icon button must have `aria-label="{Panel Name}"` so screen readers and browser native tooltips surface the panel name even without the custom tooltip. |
| Header connection dot | Green pulsing dot (2s infinite) when WebSocket connected. Red static dot when disconnected. |
| Status bar | Always visible. Updates reactively when graph data refreshes. |
| Drawer open | Slide in from right edge, 480px wide, 300ms ease-out. Semi-transparent scrim overlay on background panel (`rgba(0,0,0,0.5)`). Click scrim or press Escape to dismiss. |
| Search (type-ahead) | Debounced at 150ms. Substring match against file paths from graph node list. Show max 10 results in dropdown. Arrow keys to navigate, Enter to select. |
| `prefers-reduced-motion` | Disable: pulse animation on connection dot, FA2 layout animation, hover transitions. Keep: instant state changes, static indicators. |

### Graph Panel (VIZ-02)

| Interaction | Behavior |
|-------------|----------|
| Zoom | Mouse scroll wheel. Pinch on trackpad. Bounded: min 0.1x, max 10x. |
| Pan | Click and drag on background. |
| Click node | Open detail panel (right side overlay, not drawer). Shows: file path, centrality score, community ID, import count, importer count. |
| Hover node | Highlight node + all connected edges. Dim unconnected nodes to 20% opacity. Show tooltip with file name. |
| Hover edge | Highlight edge, show source and target file names. |
| Search bar | Top of graph panel. Type-ahead to find node by file path. Selecting a result centers and zooms to that node, highlights it. |
| Double-click node | Cross-panel link: switch to Blast Radius panel with this file pre-selected. |
| Community super-nodes (2000+) | Click super-node to drill into community subgraph. Breadcrumb shows "All > Community N". Click "All" to zoom back out. |

### Heatmap Panel (VIZ-03)

| Interaction | Behavior |
|-------------|----------|
| Click file block | Open drawer with convention detail: list of conventions, pass/fail per convention, evidence snippets. |
| Hover file block | Tooltip: file path, compliance %, violation count. |
| Collapse/expand directory | Click directory header to toggle. Collapsed shows directory-level summary bar. |
| Sort toggle | Dropdown or button group: "Worst first" (default), "A-Z", "By directory". |

### Trends Panel (VIZ-04)

| Interaction | Behavior |
|-------------|----------|
| Gauge display | Four semicircular SVG arcs. Arc fills from left to right proportional to percentage. Center text shows percentage + letter grade. |
| Trend line chart | Below gauges. Line chart with 4 series (one per dimension). |
| Legend toggle | Click legend item to show/hide that series on the chart. |
| Hover data point | Tooltip with exact value + timestamp. |

### Blast Radius Panel (VIZ-05)

| Interaction | Behavior |
|-------------|----------|
| File selection | Type-ahead search bar at top. Also auto-populated when arriving from Graph panel double-click. |
| Ring visualization | Concentric rings at hop 1, 2, 3, 4 from center. Nodes on rings sized by centrality, colored by risk. |
| Hover/click ring node | Tooltip/detail: file path, hop distance, risk level, in-degree, community. |
| Direction toggle | Button: "Dependents" (default, forward blast) / "Dependencies" (reverse). |

### Command Center Panel (VIZ-07)

| Interaction | Behavior |
|-------------|----------|
| "Run Review" card | Click opens file selector (type-ahead). After selection, triggers review analysis. Results shown in 480px right drawer. |
| "Predict Impact" card | Click opens file selector. After selection, switches to Graph panel with blast radius nodes highlighted with pulsing rings. Floating badge shows affected count + max risk. |
| "Refresh Graph" card | Click triggers incremental reparse. Shows spinner during refresh, updates status bar counts on completion. |
| "Export Screenshot" card | Click captures current panel as PNG via `html2canvas`, triggers browser download. |

### WebSocket Events (VIZ-06)

| Event | UI Response |
|-------|-------------|
| `bootstrap:progress` | Show progress banner at top of current panel: stage name + percentage bar. |
| `orient:phase` | Update progress banner with phase name. |
| `agent:spawn` | Log entry in progress banner (agent name). |
| `agent:complete` | Update log entry with completion status. |
| `graph:updated` | Auto-fetch fresh data from `/api/graph`, re-render current panel. |
| `readiness:snapshot` | Auto-fetch fresh data from `/api/readiness`, update gauges if on Trends panel. |
| Connection lost | Header dot turns red. Banner: "Connection lost. Attempting to reconnect..." Auto-reconnect: 1s, 2s, 4s, ..., max 30s backoff. |
| Reconnected | Fetch all API data to catch up. Header dot returns to green pulsing. Dismiss reconnect banner. |

## Effects & Animation

| Effect | Spec | Condition |
|--------|------|-----------|
| Connection indicator pulse | `animation: pulse 2s ease-in-out infinite` -- opacity 1.0 to 0.4 and back | WebSocket connected |
| Hover transitions | `transition: all 200ms ease-out` on interactive elements | All hover states |
| Drawer slide-in | `transform: translateX(100%) -> translateX(0)` over 300ms ease-out | Drawer open |
| Drawer slide-out | `transform: translateX(0) -> translateX(100%)` over 200ms ease-in | Drawer close |
| Accent glow | `text-shadow: 0 0 8px #22C55E` | Accent text elements only (gauge values, active metric) |
| FA2 layout | graphology-layout-forceatlas2/worker continuous -- settles naturally | Graph panel active |
| Node highlight | Connected edges brighten, unconnected nodes dim to 20% opacity | Hover on graph node |
| Reduced motion | All animations disabled except instant state changes | `prefers-reduced-motion: reduce` |

## Component Inventory

Vanilla JS modules -- no framework components. Each module exports a `render(container: HTMLElement)` function and manages its own DOM lifecycle.

### Shared Components (src/dashboard/client/components/)

| Module | Purpose | Key Elements |
|--------|---------|--------------|
| `sidebar.ts` | Navigation sidebar | 5 icon buttons (each with `aria-label`), active indicator bar, keyboard shortcut binding, tooltip on hover at all widths |
| `status-bar.ts` | Bottom status bar | Node/edge/community counts, bootstrap date, readiness grade |
| `modal.ts` | Modal overlay | Centered modal for confirmations (if needed in future) |
| `drawer.ts` | Right slide-out drawer | 480px panel for convention details, review results. Scrim overlay + Escape to close. |
| `search.ts` | Type-ahead search | Debounced input, dropdown results, keyboard navigation (up/down/enter) |
| `progress-banner.ts` | WebSocket progress overlay | Non-blocking top banner with stage name, progress bar, dismiss button |
| `tooltip.ts` | Hover tooltip | Positioned tooltip with text content, 200ms show delay |

### Panel Modules (src/dashboard/client/panels/)

| Module | Panels | Key Dependencies |
|--------|--------|------------------|
| `graph.ts` | Dependency Graph | sigma.js v3, graphology, graphology-layout-forceatlas2/worker |

## Accessibility Contract

| Requirement | Implementation |
|-------------|----------------|
| Color is not the only indicator | All colored elements (heatmap, gauges, risk levels) include numerical values or text labels alongside color |
| Keyboard navigation | Panels switchable via 1-5 keys. Drawer dismissable via Escape. Search navigable via arrow keys + Enter. |
| Focus indicators | All interactive elements show `outline: 2px solid #3B82F6` on `:focus-visible` |
| Reduced motion | `prefers-reduced-motion: reduce` disables all animations (pulse, transitions, FA2 animation). Static alternatives provided. |
| Minimum contrast | Primary text `#F8FAFC` on `#0F172A` = 15.4:1 ratio. Muted text `#94A3B8` on `#0F172A` = 5.6:1 ratio. Both exceed WCAG AA (4.5:1). |
| Screen reader | Panel switches announced via `aria-live` region. Graph node details in drawer are focusable. Status bar uses `role="status"`. Sidebar icon buttons use `aria-label="{Panel Name}"` for screen reader identification. |
| Minimum target size | All clickable elements minimum 32x32px (WCAG 2.2 Target Size minimum for non-inline targets) |
| Sidebar tooltips | Tooltip on hover retained at all viewport widths including collapsed sidebar (<1200px). `aria-label` on each icon button ensures panel names are always discoverable by assistive technology. |

**Source:** CONTEXT.md D-25 (numerical values alongside color), D-04 (reduced motion), D-18 (sidebar collapse), defaults for accessibility
