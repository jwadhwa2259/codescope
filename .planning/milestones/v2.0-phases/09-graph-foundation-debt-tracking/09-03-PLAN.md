    **1. Create `src/graph/readiness-history.ts`:**

    Import `DatabaseType` from `./database.js`, `ReadinessScore` from `../bootstrap/readiness.js`.

    Define and export `ReadinessSnapshot` interface:
    ```typescript
    export interface ReadinessSnapshot {
      id: number;
      timestamp: string;       // ISO 8601
      overall_grade: string;
      overall_percent: number;
      convention_coverage: number;
      type_safety: number;
      test_coverage_proxy: number;
      import_graph_health: number;
    }
    ```

    `storeReadinessSnapshot(db: DatabaseType, score: ReadinessScore): void`:
    - Prepare and run:
      ```sql
      INSERT INTO readiness_history
        (timestamp, overall_grade, overall_percent, convention_coverage, type_safety, test_coverage_proxy, import_graph_health)
      VALUES (?, ?, ?, ?, ?, ?, ?)
      ```
    - Values: `new Date().toISOString()`, `score.overall.grade`, `score.overall.percent`, `score.dimensions.conventionCoverage.percent`, `score.dimensions.typeSafety.percent`, `score.dimensions.testCoverageProxy.percent`, `score.dimensions.importGraphHealth.percent`

    `getLatestSnapshot(db: DatabaseType): ReadinessSnapshot | null`:
    - Query: `SELECT * FROM readiness_history ORDER BY timestamp DESC LIMIT 1`
    - Return the row cast to `ReadinessSnapshot`, or `null` if no rows

    `getSnapshotNear(db: DatabaseType, targetTimestamp: string): ReadinessSnapshot | null`:
    - Query: `SELECT * FROM readiness_history WHERE timestamp <= ? ORDER BY timestamp DESC LIMIT 1`
    - Parameter: `targetTimestamp` (ISO 8601 string)
    - Return the row cast to `ReadinessSnapshot`, or `null` if no rows before target

    **1. Create `src/tools/trends-tool.ts`:**

    Import: `McpServer` from MCP SDK, `isBootstrapped`, `errorResponse`, `okResponse`, `buildMetadata` from `./helpers.js`, `openDatabase`, `closeDatabase` from `../graph/database.js`, `getGraphDbPath` from `../utils/paths.js`, `getLatestSnapshot`, `getSnapshotNear`, `ReadinessSnapshot` from `../graph/readiness-history.js`.

    Define and export `trendDirection(current: number, previous: number): 'improving' | 'declining' | 'stable'`:
    - `const delta = current - previous`
    - If `Math.abs(delta) <= 1` return `'stable'` (within 1% is noise)
    - If `delta > 0` return `'improving'`
    - Return `'declining'`

    Define a `TrendComparison` interface:
    ```typescript
    interface TrendComparison {
      label: string;                        // "vs_previous", "vs_7_days_ago", "vs_30_days_ago"
      snapshot: ReadinessSnapshot | null;   // The comparison snapshot
      deltas: {
        overall_percent: number;
        convention_coverage: number;
        type_safety: number;
        test_coverage_proxy: number;
        import_graph_health: number;
      } | null;
      trend: 'improving' | 'declining' | 'stable' | null;
    }
    ```

    Helper function `buildComparison(label: string, current: ReadinessSnapshot, other: ReadinessSnapshot | null): TrendComparison`:
    - If `other` is null, return `{ label, snapshot: null, deltas: null, trend: null }`
    - Compute deltas: `current.X - other.X` for each numeric field
    - Compute overall trend: `trendDirection(current.overall_percent, other.overall_percent)`
    - Return the full comparison object

    Export `handleTrends(projectRoot: string): Promise<{ content: Array<{ type: "text"; text: string }> }>`:
    - `const startMs = Date.now()`
    - Guard: `if (!isBootstrapped(projectRoot))` return errorResponse("NOT_BOOTSTRAPPED", ...)
    - Open database: `const db = openDatabase(getGraphDbPath(projectRoot))`
    - Wrap in try/finally with `closeDatabase(db)`:
      - `const current = getLatestSnapshot(db)`
      - If `current` is null: return `errorResponse("NO_HISTORY", "No readiness history found. Run /codescope:bootstrap first.", "Run /codescope:bootstrap to start tracking readiness trends.")`
      - Get previous snapshot: query `SELECT * FROM readiness_history ORDER BY timestamp DESC LIMIT 1 OFFSET 1` (use db.prepare directly for this one -- or add a `getPreviousSnapshot` helper to readiness-history.ts)
      - Get 7-day-ago snapshot: `getSnapshotNear(db, new Date(Date.now() - 7 * 24 * 60 * 60 * 1000).toISOString())`
      - Get 30-day-ago snapshot: `getSnapshotNear(db, new Date(Date.now() - 30 * 24 * 60 * 60 * 1000).toISOString())`
      - Build comparisons: `buildComparison('vs_previous', current, previous)`, `buildComparison('vs_7_days_ago', current, weekAgo)`, `buildComparison('vs_30_days_ago', current, monthAgo)`
      - Get total snapshot count: `SELECT COUNT(*) as count FROM readiness_history`
      - Build response data:
        ```typescript
        {
          current: {
            timestamp: current.timestamp,
            overall_grade: current.overall_grade,
            overall_percent: current.overall_percent,
            dimensions: {
              convention_coverage: current.convention_coverage,
              type_safety: current.type_safety,
              test_coverage_proxy: current.test_coverage_proxy,
              import_graph_health: current.import_graph_health,
            }
          },
          comparisons: [previousComparison, weekComparison, monthComparison],
          snapshot_count: count,
        }
        ```
      - Return `okResponse(data, buildMetadata(projectRoot, startMs))`

    Export `registerTrendsTool(server: McpServer, projectRoot: string): void`:
    - Call `server.tool(...)` with:
      - Name: `"codescope_trends"`
      - Description: `"Get readiness trend data showing how your codebase health changes over time. Returns the current readiness snapshot with three comparisons: vs. previous snapshot, vs. 7 days ago, vs. 30 days ago. Each comparison includes per-dimension deltas and trend direction (improving/declining/stable). Related tools: codescope_readiness."`
      - Input schema: `{}` (no parameters needed)
      - Handler: `async () => handleTrends(projectRoot)`
