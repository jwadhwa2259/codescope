    - Register `"codescope_detect_changes"` with description per D-35: "Detect changed files and classify their risk based on knowledge graph centrality.     - Input schema: `{ diff: z.string().optional().describe("Git diff output to analyze. If not provided, uses current working directory changes.") }`
    - Handler per D-23, D-24:

      2. Get changed files: if `diff` param provided, parse file paths from it (lines starting with `+++ b/` or `diff --git`). If not provided, run `execSync("git diff --name-only HEAD", { cwd: projectRoot })` to get working directory changes.

      4. For each changed file:
         a. Find file node in graph: `graph.filterNodes((n, attr) => attr.filePath === filePath && attr.kind === "file")`

         d. Get blast_radius_count: call `blastRadius(graph, nodeId, 4).length` per D-24
         e. If file not in graph, classify as LOW with blast_radius_count: 0
      5. Return okResponse with `{ changed_files: Array<{ path, risk, centrality, blast_radius_count }>, summary: { total, high, medium, low } }` and metadata
