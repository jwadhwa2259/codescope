    - Export `interface ForceConfirmation { willRebuild: string[]; willPreserve: string[]; }`
    - Export `function getForceConfirmation(projectRoot: string): ForceConfirmation` -- returns lists of what --force will rebuild (graph.db, service artifacts, readiness.md, service-manifest.md, cross-service-map.md, danger-zones.md) and what it will preserve (config.yml, conventions-enforced.md, learnings.md, orient/, plans/, execution/, reports/) per D-29
    - Export `interface BootstrapOptions { projectRoot: string; force?: boolean; maxSquads?: number; onProgress?: (message: string) => void; onConfirm?: (confirmation: ForceConfirmation) => Promise<boolean>; }`
    - Export `interface BootstrapResult { services: Array<{ name: string; status: "full" | "lightweight"; durationMs: number }>; readinessGrade: string; readinessPercent: number; totalNodes: number; totalEdges: number; totalCommunities: number; conventionsDetected: number; highConfidenceConventions: number; durationMs: number; artifacts: Array<{ path: string; description: string }>; timingBreakdown: Record<string, number>; warnings: string[]; }`
    - Export `async function runBootstrap(options: BootstrapOptions): Promise<BootstrapResult>`:

         b. Else call `analyzeChanges(projectRoot, readBootstrapMeta(projectRoot)?.last_bootstrap ?? null, totalFileCount, services)`

      4. Report progress per D-06: `onProgress("## Scanning services...")`

      16. Compute timing breakdown per D-31 from agent durations
      17. If total duration > 300000ms (5 min), add warning per D-32
      18. Return BootstrapResult with all fields
