    **1. Create `src/graph/migration.ts`:**

    ```typescript
    export const CURRENT_SCHEMA_VERSION = 2;
    ```

    Export `migrateDatabase(db: DatabaseType, dbPath: string): void`:
    - Read current version: `db.pragma('user_version', { simple: true }) as number`
    - If `currentVersion >= CURRENT_SCHEMA_VERSION`, return immediately (already up to date)
    - If `currentVersion < 2`, call `migrateToV2(db)` inside a try/catch:
      - On success: `db.pragma('user_version = 2')`
      - On failure: close the db, delete the file at `dbPath`, throw a specific error `new Error('MIGRATION_FAILED')` so the caller can handle fallback
    - `migrateToV2(db)`: Execute the SQLite 12-step table recreation process per D-06/D-07:
      - Disable foreign keys: `db.pragma('foreign_keys = OFF')`
      - Wrap in `db.transaction(...)`:
        - Create `new_edges` table with ON DELETE CASCADE on source_id and target_id (same columns as current edges: id INTEGER PRIMARY KEY, source_id INTEGER REFERENCES nodes(id) ON DELETE CASCADE, target_id INTEGER REFERENCES nodes(id) ON DELETE CASCADE, kind TEXT NOT NULL, weight REAL DEFAULT 1.0, metadata JSON)
        - `INSERT INTO new_edges SELECT * FROM edges`
        - `DROP TABLE edges`
        - `ALTER TABLE new_edges RENAME TO edges`
        - Recreate indexes: `idx_edges_source` on edges(source_id), `idx_edges_target` on edges(target_id), `idx_edges_kind` on edges(kind)
        - Create `new_communities` table with ON DELETE CASCADE (node_id INTEGER REFERENCES nodes(id) ON DELETE CASCADE, community_id INTEGER, modularity_class TEXT)
        - `INSERT INTO new_communities SELECT * FROM communities`
        - `DROP TABLE communities`
        - `ALTER TABLE new_communities RENAME TO communities`
        - Create `file_hashes` table: `CREATE TABLE IF NOT EXISTS file_hashes (file_path TEXT PRIMARY KEY, content_hash TEXT NOT NULL, updated_at INTEGER NOT NULL)`
        - Create `readiness_history` table: `CREATE TABLE IF NOT EXISTS readiness_history (id INTEGER PRIMARY KEY, timestamp TEXT NOT NULL, overall_grade TEXT NOT NULL, overall_percent INTEGER NOT NULL, convention_coverage INTEGER NOT NULL, type_safety INTEGER NOT NULL, test_coverage_proxy INTEGER NOT NULL, import_graph_health INTEGER NOT NULL)`
        - Create index: `CREATE INDEX IF NOT EXISTS idx_readiness_ts ON readiness_history(timestamp)`
      - Re-enable foreign keys: `db.pragma('foreign_keys = ON')`
      - Validate: `const fkCheck = db.pragma('foreign_key_check') as unknown[]`; if non-empty, throw
