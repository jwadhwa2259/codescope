      - If no stored hash exists (new file, lazy population): push to stale array
      - If stored hash differs from current hash: push to stale array
      - If hashes match: skip (file is fresh)
    - Return stale array

    - Use `INSERT OR REPLACE INTO file_hashes (file_path, content_hash, updated_at) VALUES (?, ?, ?)`
