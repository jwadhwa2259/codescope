<objective>
Build the session continuity core: types, handoff document generator from pipeline state, handoff parser for resume, and 7-day session file cleanup.

Purpose: This creates the structured handoff documents that capture where a CodeScope pipeline stopped. The generator reads existing filesystem artifacts (scope contracts, plans, coordination logs) and produces a markdown document with YAML frontmatter. The parser reads that document back for resume. Plan 04 will wire these into Claude Code hooks and skills.

Output: `src/session/` module with types, generator, parser, and cleanup; tests proving roundtrip fidelity and cleanup behavior.
