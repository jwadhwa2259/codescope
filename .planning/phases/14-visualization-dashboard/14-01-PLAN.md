**Create `src/dashboard/api/graph.ts`:**

Route handler for `GET /graph`:
- Get `projectRoot` from `c.get('projectRoot')`
- Get `dbPath` from `getGraphDbPath(projectRoot)`
- If db file doesn't exist, return `c.json({ status: 'error', code: 'NOT_BOOTSTRAPPED', message: 'No codebase data yet' }, 404)`
- Open db with `openDatabase(dbPath)` in try/finally (closeDatabase in finally)
- Call `loadGraphFromSQLite(db)`, `computeCentrality(graph)`, `runCommunityDetection(graph)`, `computeDangerZones(graph, centralities, communities.communities)`
- Build danger zone set from `dangerZones.map(d => d.filePath)`
- Serialize nodes: `graph.forEachNode((id, attrs) => nodes.push({ id, ...attrs, centrality: centralities.get(id) ?? 0, community: communities.communities.get(id) ?? 0, isDangerZone: dangerZoneSet.has(attrs.filePath) }))`
- Serialize edges: `graph.forEachEdge((id, attrs, source, target) => edges.push({ id, source, target, ...attrs }))`
- Pre-compute circular layout positions grouped by community: for each community group, arrange nodes in a circle segment so the graph is readable on first render (per D-20)
- Return `c.json({ status: 'ok', data: { nodes, edges, communities: communityMap, dangerZones } })`

**Create `src/dashboard/api/conventions.ts`:**

Route handler for `GET /conventions`:
- Read convention index JSON from `.claude/codescope/injection/convention-index.json` via `getCodescopePath(projectRoot)`
- If file doesn't exist, return 404 with `code: 'NO_CONVENTIONS'`
- Parse and return the `ConventionIndex` data

**Create `src/dashboard/api/readiness.ts`:**

Route handler for `GET /readiness`:
- Open db, get latest snapshot via `getLatestSnapshot(db)`
- Query all history: `db.prepare('SELECT * FROM readiness_history ORDER BY timestamp ASC').all()`
- Compute letter grades for each dimension: A (>=90), B (>=75), C (>=60), D (>=45), F (<45)
- Return `c.json({ status: 'ok', data: { current: latestWithGrades, history } })`

**Create `src/dashboard/api/blast-radius.ts`:**

Route handler for `GET /blast-radius/:file`:
- Decode file path from URL param: `decodeURIComponent(c.req.param('file'))`
- Direction query param: `c.req.query('direction') ?? 'forward'`
- Open db, load graph, compute centrality
- If direction is 'forward', call `blastRadius(graph, centralities, filePath, 4)`
- If direction is 'reverse', call `reverseBlastRadius(graph, centralities, filePath, 4)`
- Group results by hop distance for ring visualization
- Return `c.json({ status: 'ok', data: { file: filePath, direction, rings, totalAffected } })`

**Create `src/dashboard/api/status.ts`:**

Route handler for `GET /status`:
- Open db, count nodes: `SELECT COUNT(*) as count FROM nodes`
- Count edges: `SELECT COUNT(*) as count FROM edges`
- Count communities: `SELECT COUNT(DISTINCT community_id) as count FROM communities`
- Get bootstrap meta from `.claude/codescope/meta.json` if exists
- Return `c.json({ status: 'ok', data: { nodeCount, edgeCount, communityCount, bootstrapDate, isBootstrapped } })`

**Create `src/dashboard/api/review.ts`:**
